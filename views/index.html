File Metadata Microservice

FreeCodeCamp APIs & Microservices Project

The File Metadata app consists of a form that lets the user upload an input file upon submission and see its filesize. In particular:

  • The user can submit a form that includes a file for upload.
  • The input field of the form file have the attribute "name" set to "upfile".
  • Submission of a file generates a JSON response of the form:
    {name: file name, type: file type, size: file size in bytes}
  • To handle the file uploading the multer npm package is used ( https://www.npmjs.com/package/multer).
